# DRH Portfolio

Personal portfolio site for Dakota Harmon — photographer and web developer currently based in Lubbock, Texas.

**Live site:** [drh-creates.github.io/portfolio](https://drh-creates.github.io/portfolio)

---

## About

A single-page portfolio built with vanilla HTML, CSS, and JavaScript. No frameworks, no dependencies beyond Google Fonts and Font Awesome.

## Sections

- **Photography** — gallery of selected work
- **Web Design** — web design projects with live and source links
- **About** — background and skills
- **Contact** — links and contact form

## Tech

- HTML5 / CSS3 / Vanilla JS
- Google Fonts (Babylonica, Quintessential, Truculenta, JetBrains Mono)
- Font Awesome 6
